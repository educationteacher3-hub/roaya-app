# 📊 مكتب رؤية — النظام المالي

نظام عرض تقارير مالي يقرأ مباشرة من ملفات Excel الخاصة بك.

---

## 🗂️ هيكل الملفات

```
roaya_app/
│
├── app.py                          ← الصفحة الرئيسية (لوحة التحكم)
├── data_loader.py                  ← محرك قراءة ملفات Excel
├── requirements.txt                ← المكتبات المطلوبة
├── تشغيل.bat                       ← ملف التشغيل (اضغط عليه مباشرة)
│
├── pages/
│   ├── 1_خزينة.py                 ← حركة الخزينة
│   ├── 2_عملاء_وموردين.py         ← العملاء والموردين والفواتير
│   ├── 3_تقارير.py                ← قائمة الدخل وتحليل المصروفات
│   └── 4_اتقان.py                 ← كشف حساب اتقان (الإمارات)
│
└── data/                           ← ← ← ضع ملفاتك هنا
    ├── ROAYA__CASH.xlsx
    ├── حسابات_العملاء_والموردين.xlsx
    └── اتقان_09_6_2026.xlsx
```

---

## 🚀 خطوات التشغيل

### الخطوة 1: تثبيت Python
- حمّل Python من: https://www.python.org/downloads/
- أثناء التثبيت: **فعّل خيار "Add Python to PATH"** ✅

### الخطوة 2: وضع الملفات
- انسخ ملفاتك الثلاثة داخل مجلد `data/`

### الخطوة 3: التشغيل
- **اضغط مرتين** على ملف `تشغيل.bat`
- ستفتح نافذة سوداء وتثبّت المكتبات تلقائياً (مرة واحدة فقط)
- يفتح المتصفح تلقائياً على: `http://localhost:8501`

---

## 🔄 تحديث البيانات

- عدّل ملفات Excel كما تريد
- اضغط زر **"↻ تحديث البيانات"** في التطبيق
- أو اضغط `F5` في المتصفح

---

## ⚠️ ملاحظات مهمة

1. **أسماء الشيتات**: التطبيق يعتمد على أسماء محددة:
   - `ROAYA__CASH.xlsx` → شيتات: `حركة الخزينة`, `قائمة الدخل`, `تحليل مصروفات شهري`
   - `حسابات_العملاء_والموردين.xlsx` → شيتات: `مستر`, `العملاء`, `الموردين`, `قائمة الموردين والعملاء`
   - `اتقان_*.xlsx` → شيت: `كشف الحساب`

2. **لو غيّرت اسم ملف اتقان**: عدّل المسار في `data_loader.py` السطر رقم 12

3. **مشكلة في البيانات؟** تأكد إن الهيدر في نفس الصف اللي التطبيق متوقعه

---

## 🛠️ تخصيص

لإضافة عملاء أو موردين جدد: أضفهم في ملف Excel مباشرة ثم اضغط تحديث

لإضافة صفحة جديدة: أنشئ ملف `.py` جديد في مجلد `pages/`
